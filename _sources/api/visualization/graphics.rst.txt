Graphics
======== 
 
.. autopostdoc:: ansys.fluent.visualization.pyvista.pyvista_objects.Graphics   

In the following example, a Graphics object is instantiated with a Fluent session as its context. The Graphics object is used to create mesh, contour and surface objects. 

        from ansys.fluent.visualization.pyvista import  Graphics
        
        graphics_session1 = Graphics(session)
        mesh1 = graphics_session1.Meshes["mesh-1"]
        contour1 = graphics_session1.Contours["contour-1"]
        contour2 = graphics_session1.Contours["contour-2"]
        surface1 = graphics_session1.Surfaces["surface-1"]
        del graphics_session1.Contours["contour-1"]
